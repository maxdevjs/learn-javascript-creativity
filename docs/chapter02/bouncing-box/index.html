This is a bouncing box example.

There is a bit of trigonometry involved to take an angle (it will begin as a random angle under 360 degrees) and to find the desired position (heading of box).

const degreesToRadians = degrees => degrees * (Math.PI / 180);

const dir_x = (length, angle) => length * Math.cos(degreesToRadians(angle));

const dir_y = (length, angle) => length * Math.sin(degreesToRadians(angle));

duration is zeroed when equal to 10.
